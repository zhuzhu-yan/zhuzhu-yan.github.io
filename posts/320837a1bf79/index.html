现在在Vs Code,在这里编辑，编辑完成之后，

  1. 保存（ctrl+s);
  2. 暂存：在终端（命令行）进行暂存:git add .此时也可查看仓库的状态：git status
  3. 提交：git commit -m 编辑博客
  4. 推送：git push
